How to Play

Objective

  • Explore the world
  • Find keys scattered across the map and place them in "the zone"
  • Enter the portal to win
  • Collect orbs to use abilities
  • Avoid "the entity"

Main Menu

  • n: Start new game
  • l: Load game
  • q: Quit

In Game

  • Move: w, a, s, d
  • j: Pick up / place key
  • k: Dash
  • m: Teleport

Other Keys

  • z: Toggle lights
  • x: Show path
  • v: Save
  • l: Load
  • r: Restart
  • : then q: Quit

HUD

  • Facing direction
  • Number of keys placed
  • Orbs collected
  • Tile under cursor
