AI Multiverse - Voice-Driven 3D AI Sandbox

AI Multiverse is a real-time 3D multiplayer world built in Godot where you can walk up to AI characters (Obama, Spongebob, etc.) and talk to them naturally by voice. Player audio is streamed through a Python sidecar, transcribed live, passed to an LLM for response, and played back in-game with text-to-speech. We also built turn-taking and smart interruption logic so characters don't all talk over each other. The system supports positional voice chat, so conversations feel like proximity-based NPCs in a game, not a chat app.

  • Godot game engine + GDScript for multiplayer state, movement, and world logic
  • Python WebSocket sidecar for speech-to-text and model inference
  • Character "conversation manager" for timing, handoffs, and interruption control
  • Designed as an always-on social space where players and AI share the same world

Download the game (.jar)

To run:

  1. Make sure you have Java installed (JDK or JRE).
  2. Download the .jar file from the link above.
  3. Open a terminal and navigate to the download location.
  4. Run the following command: java -jar lights-off.jar
  5. In the game: press n for a new game, type a numeric seed, then press s to create the world.
How to Play
  • Explore the world
  • Find keys scattered across the map and place them in "the zone"
  • Enter the portal to win
  • Collect orbs to use abilities
  • Avoid "the entity"
  • n: Start new game
  • l: Load game
  • q: Quit
  • Move: w, a, s, d
  • j: Pick up / place key
  • k: Dash
  • m: Teleport
  • z: Toggle lights
  • x: Show path
  • v: Save
  • l: Load
  • r: Restart
  • : then q: Quit
  • HUD: Facing, keys, orbs, cursor tile
