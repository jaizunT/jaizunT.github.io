Motivation: Orthogonalization for Muon Optimization

In modern deep learning, optimizers like Muon achieve state-of-the-art performance by orthogonalizing gradient updates during model training. Mathematically, this step normalizes the update geometry, forcing singular values of the gradient matrix toward 1.

The idea: a full SVD each step is too expensive; iterative methods like Polar Express use only matrix–matrix multiplies (GPU-friendly) to “flatten” the spectrum without explicit decomposition.

The Algorithms Visualized

Polar Express

A hardware-efficient, polynomially scaled iteration that speeds singular-value convergence with stability parameters (“cushion”/“safety”).

Newton–Schulz

Classic polar iteration with quadratic local convergence: X_{k+1} = 0.5 * X_k (3I - X_k^T X_k).

Matrix Sign (Jordan)

Robust/global but typically needs inversion: X_{k+1} = 0.5 * (X_k + X_k^{-T}).
