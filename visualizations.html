Polar Express - SVD Visualization

An interactive web application that visualizes the evolution of singular values during the Polar Express algorithm. Users can adjust parameters, draw custom initial distributions by hand, and watch how the singular values converge in real-time.

  • Tech Stack: React, Vite, Tailwind CSS, Plotly.js
  • Interactions: Sketching canvas, 3D density landscape, Real-time parameter tuning
